Privacy Policy

Last updated: 30 May 2026

This Privacy Policy describes how the personal, single-user YouTube niche-research tool ("the Tool") handles data. The Tool is used solely by its owner to research publicly available YouTube video performance in order to plan the owner's own video content. It is not a public service, has no other users, and is not monetized.

1. Data we collect

The Tool collects no personal data and has no user accounts, sign-up, login, tracking, cookies, or analytics. It does not ask anyone to authenticate with Google credentials.

2. YouTube data we access

The Tool accesses publicly available data through the official YouTube Data API v3 only — specifically public video statistics (view counts), public channel statistics (subscriber counts, upload lists), and video/channel titles and descriptions. No private or user-authorized (OAuth) data is accessed. No scraping or unofficial endpoints are used.

3. How the data is used

The retrieved public statistics are displayed to the owner and used to compute a simple arithmetic ratio (a video's view count divided by the average view count of its channel's recent uploads) to help the owner identify which topics perform well. This ratio is computed locally and is clearly labeled as a local calculation, not a metric provided by YouTube.

4. Data storage and retention

Public YouTube API data is cached locally on the owner's machine with a fetch timestamp and is automatically deleted after a maximum of 30 calendar days, after which it is re-fetched if still needed. No data is stored on any server or shared cloud.

5. Data sharing

No data is sold, shared, or transferred to any third party. Data never leaves the owner's local environment.

6. Your YouTube data and rights

This Tool uses YouTube API Services. By using it, you are agreeing to be bound by the YouTube Terms of Service. Data handling by Google is governed by the Google Privacy Policy. You can manage or revoke the Tool's access to Google/YouTube data via the Google security settings. Any stored data will be deleted within 30 days of a deletion request.
